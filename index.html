— 07 / DÚVIDAS

Perguntas frequentes.

Preciso instalar .NET ou outra coisa? +

Não. O Ultra DICOM já vem com tudo embutido em um único arquivo .exe. Basta dar duplo clique em qualquer Windows 10 ou 11.

Funciona com qualquer equipamento médico? +

Sim, suportamos os padrões DICOM Basic Grayscale Print Management e Basic Color Print Management — usados praticamente por todos os equipamentos modernos: GE, Siemens, Philips, Toshiba, Mindray, Samsung, Canon Medical e outros.

Posso instalar a mesma licença em mais de uma máquina? +

Não. Cada licença é amarrada ao hardware da máquina onde foi ativada. Para usar em outro PC, é necessária uma nova licença ou solicitação de transferência (em casos de troca de equipamento).

O que acontece quando a licença expira? +

O programa solicita uma nova chave de ativação para continuar funcionando. Suas configurações ficam salvas — basta renovar e voltar a imprimir.

Como recebo a chave de ativação após pagar? +

Após o pagamento, envie por e-mail ou WhatsApp o Hardware ID exibido pelo app. Em até 1 hora útil você recebe sua chave personalizada.

Funciona offline? +

Sim. Após a ativação inicial, todo o processamento de impressão é local. Internet só é necessária para baixar o programa e na renovação anual.
